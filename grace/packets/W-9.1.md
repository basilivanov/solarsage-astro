---

## Decision

Create microcopy system with:
1. **Schema** — JSON schema defining microcopy structure (keys, categories, variants)
2. **Dictionary v1** — initial microcopy content (UI labels, CTAs, error messages)
3. **Microcopy service** — Python service for loading and serving microcopy

**Why:** Centralized microcopy enables content updates without code changes. Schema ensures consistency. Service provides type-safe access to copy strings.
