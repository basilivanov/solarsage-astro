status: completed

## Decision

Create LLM service with:
1. **Prompt v1** — system prompt template for astrological reading generation
2. **Context builder** — assembles semantic blocks into LLM-ready context
3. **JSON validator** — ensures LLM output matches `TodayPayload` schema

**Why:** LLM needs structured prompts and validated outputs. Context builder transforms semantic data into natural language context. JSON validator prevents malformed responses from reaching frontend.
