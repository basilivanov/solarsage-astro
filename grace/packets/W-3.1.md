## Decision

Create FastAPI sidecar service (`apps/solarsage/solarsage/app.py`) with basic scaffolding and health endpoint. This service will wrap the reference collector and provide HTTP API for astrological calculations.

**Why:** Separate the calculation engine from the main API to enable independent scaling and testing. Health endpoint establishes the service contract.
